Template-driven Forms

  • ngModel name="controlName" in input template driven aproach
  • #templateVar="ngModel" in input reference to this ngModel
  • #formName="ngForm" in form to get reference
  • (ngSubmit)="submitFn(formName)"onSubmit function
  • [ngValue]="object" use to set the value as an object.
