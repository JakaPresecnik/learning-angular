Displaying Data and Handling Events

Installed bootstrap and bootstrap-icons

  • [class.active]="boolean" for class binding
  • (click)="myFunction($event)" for event binding
    • $event.stopPropagation() inside myFunction to stop event bubbling
    • *$event.target.value* inside myFunction to get an input value
  • instead of * use #name in input and pass name.value into myFunction
  • For two way binding (for form inputs):
    • use [(ngModel)]="name"
    • and register FormsModule in module.ts inside imports
  • ng g pipe pipeName for costum pipes
  • if(!value) return null use inside transform, add value.something after

Building Re-usable Components

  • @Input() prop: boolean; use this decorator to bind properties
  • @Output() change = new EventEmitter() use output prop to raise event in this
    • this.change.emit(this.prop) inside a click or other function in that comp
  • <ng-content select=".class"></ng-content> in child comp, use class in app.comp
  • <ng-container"></ng-container> use to display without div,p...

Directives

  • *ngIf See link
  • [ngSwitch]="viewMode" to initialize switch on a content
    • *ngSwitchCase="'map'" to check the value to display a certain content
    • *ngSwitchDefault use to display other cases
  • *ngFor="let item of items; index as i" to get the index link
    • use trackBy: trackByIdFn to track items by Ids
  • [ngClass]="{'some-class': boolean}
  • ng g directive directive-name
